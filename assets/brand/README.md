# SAEP brand assets

Canonical, license-clean marks for the SAEP protocol.

| File | Use |
|---|---|
| `logo.svg` | Mark only — favicon, app icon, compact headers. Stroke inherits `currentColor`. |
| `wordmark.svg` | Lockup for headers and footer. Uses Archivo Black (self-host via `@fontsource/archivo-black`). |

Palette, typography, and usage rules: see the brand guide in the SAEP-internal repo.

The lime accent (`#CBFF3A`) is the only chromatic color. Use sparingly.
