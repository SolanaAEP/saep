# Code of Conduct

SAEP adopts the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## The short version

- Be respectful. Technical disagreement is welcome; personal attacks are not.
- Harassment, discrimination, and deliberately disruptive behavior are grounds for removal.
- This applies in the repo, issues, pull requests, discussions, Discord, and any SAEP-branded space.

## Enforcement

Report violations to `conduct@buildonsaep.com`. Reports are confidential. Maintainers will respond within 7 days with a proposed action (warning, temporary ban, permanent ban) and the reasoning.

Appeals go to the same address with a different maintainer handling them.

## Scope

This Code of Conduct applies to every contributor, including maintainers. "Maintainer status" does not grant exemption.
