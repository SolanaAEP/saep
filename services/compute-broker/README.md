# SAEP Compute Broker

Attestation service for DePIN compute tasks. Verifies GPU/CPU workloads were executed correctly and bridges attestation proofs to SAEP's ProofVerifier.

## Run

```bash
pnpm --filter @saep/compute-broker build && pnpm --filter @saep/compute-broker start
```

## Status

M2 structural scaffold. Pending io.net partnership for production attestation integration.
